Candies

Enjoy music on a forest train journey and color the forest with any melody you make. Make a sound with Arduino Adafruit Circuit playground to change the color of trees in the forest. Make other interactions with the photoresistor and a button.

Detail Introduction

procedural

This project focuses on applying procedural methods to autonomously generate game-level content. I will explore two main procedural techniques: graph grammar, which can be used to procedurally generate game levels, and the procedural method utilized in Spelunky, a game known for its procedural game-level design. This project aims to apply the fusion of these techniques to create procedural 2D side-scrolling game levels.
